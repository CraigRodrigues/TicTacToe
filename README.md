# CLI TicTacToe

#### Simple TicTacToe command-line game

![playthrough](http://i.imgur.com/02jAI8o.gif)

To play:

```
npm install
npm start
```

Instructions:
- Start the game
- Input player names
- The game will randomly choose who goes first
- Once someone wins or there is a draw choose to play again if you want

Testing with Tape:
```
npm test
```

*Requirements: ES6*
